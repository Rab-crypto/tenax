Features

Everything you need to preserve context

Built for developers who need their AI assistant to remember what matters.

Automatic Capture

Sessions are saved via hooks with zero friction. Every decision, pattern, and insight is captured automatically.

Semantic Search

Find anything with natural language. Search across all your project knowledge using AI-powered similarity matching.

Token Transparent

Know exactly what you're spending. Every operation shows precise token costs so you stay in control of your context budget.

Local & Private

Embeddings run entirely on your device using local models. No API calls, no data leaving your machine.

Export Anywhere

Export to Markdown, JSON, Obsidian, or Notion. Your knowledge is portable and integrates with your existing tools.

Proactive Checking

Claude automatically checks your history before proposing solutions. No more re-deciding the same things twice.

How It Works

Three steps to persistent context

Knowledge flows naturally from conversation to searchable memory.

1

Work with Claude

Discuss architecture, patterns, and solutions. Claude marks important decisions in its responses using special markers.

2

Auto-Capture

When your session ends, hooks automatically extract marked knowledge and store it with embeddings for semantic search.

3

Search & Load

Use commands like /project-memory:search to find past decisions. Claude checks history before proposing changes.

Quick Start

Up and running in seconds

Install the plugin, have a conversation with decisions, and watch your knowledge persist automatically.
